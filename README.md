## Components

- Auth pages
- Form inputs, hooks and validators
- Input Pickers
- Survey module
- Snackbar
- Tables: static & dynamic
- MUI Theme loader

Localization ready

## Get Started

To get started, you need to:

1. Installation

```shell
npm i @famalabs/map-ui
```

2. Install dependencies

```
cd <project_name>
npm install
```

3. Import ThemeLoader in your App
